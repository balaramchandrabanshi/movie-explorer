# 🎬 Movie Explorer

Movie Explorer is a React application that allows users to discover popular movies, search for movies, and view detailed information using the TMDB API.

## ✨ Features

- Browse popular movies
- Search movies by title
- View detailed movie information
- Responsive UI
- Loading indicator
- No Movies Found page
- Custom 404 Page

## 🛠️ Tech Stack

- React
- React Router
- Tailwind CSS
- TMDB API
- Vite

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/your-username/movie-explorer.git
```

2. Install dependencies

```bash
npm install
```

3. Create a `.env` file

```env
VITE_TMDB_API_TOKEN=your_tmdb_access_token
```

4. Start the development server

```bash
npm run dev
```
